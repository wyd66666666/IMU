版本：V1.0（2025\8\7）

功能说明：
       连接匿名助手可以看到传感器的数据

使用主要技术
	功能/解决的问题	                使用的算法/滤波器
	消除Roll/Pitch的持续漂移	均值滤波 (用于初始陀螺仪校准)
	消除传感器高频噪声/微抖动	动态自适应低通滤波器 (根据运动状态调整强度)
	融合多传感器数据计算姿态	Madgwick AHRS 姿态融合算法 (基于四元数)
	锁定静止时的姿态，防止微抖	动态死区处理 (Dynamic Deadzone)
	解决Yaw旋转时的数值突变	角度连续性处理 (Angle Unwrapping)
	在稳定和响应之间取得平衡	滤波模式切换 (通过按键在4种预设参数间切换)

接线图：
       主控：STM32G431CBT6
       	按键 <--------> PC13
       	LED  <--------> PC6

       BMI270+BMM350（软IIC） 
	SCL  <--------> PA15
	SDA <--------> PB7
	VCC <--------> 3.3V
	GND<--------> GND

       BMP580（软IIC） 
	SCL  <--------> PC4
	SDA <--------> PA8
	VCC <--------> 3.3V
	GND<--------> GND

       STLINK
	RST <--------> NRST
	其他四线正常接

       USB-TTL
	绿   <--------> PA9
	蓝   <--------> PA10
	GND<--------> GND

程序使用方法:
       1.连接type-c和串口
       2.按KEY进行校准
       3.灯光闪烁后说明程序正常运行，可以看上位机数据
       4.灯光闪烁后按下KEY进行模式切换，打开磁力计，程序默认不开磁力计

目前程序存在问题：
       不加磁力计上位机显示的无人机会有点抖，但是整体数据较稳
       加入磁力计后数据都飘，且YAW RLO飘的最厉害
       磁力计受干扰，手机靠近磁力计都会引起磁力计变化
